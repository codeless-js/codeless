# build-plugin-vue-iview

> The basic webpack configuration for ice project

## Usage

### registerUserConfig

`build.json`

```json
{
  "plugins": [
    ["build-plugin-vue-iview",{
      "usingTemplate": true,
      "materialType": "page"
    }]
  ]
}
```

使用方法参考queryTable物料，物料地址：https://github.com/codeless-js/vue-materials-iview/tree/master/pages/QueryTable
